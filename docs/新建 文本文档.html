健康管理系统 - 项目功能总览

一、项目结构

  • health_backend：Spring Boot 后端，提供Web接口和安全配置
  • health_service_provider：业务实现层，MyBatis-Plus，数据库操作
  • health_interface：服务接口定义
  • health_common：通用工具类、实体、常量
  • sql/：数据库脚本

二、主要功能模块

  • 体检项管理（CheckItem）
  • 体检组管理（CheckGroup）
  • 会员管理（Member）
  • 用户与权限管理（User, Role, Permission）
  • 预约设置与报表（OrderSetting, Report）

三、主要后端接口

  • /checkitem：体检项增删改查
  • /checkgroup：体检组增删改查
  • /member：会员管理
  • /user：用户管理与登录
  • /role、/permission：角色与权限分配

四、技术栈

  • 后端：Spring Boot、MyBatis-Plus、Spring Security
  • 前端：HTML、CSS、JavaScript、Vue.js、Element UI
  • 数据库：MySQL

五、项目亮点

  • 分层架构，接口与实现分离，便于维护和扩展
  • 支持角色权限管理，安全性高
  • 前后端分离，页面美观，交互友好
  • 支持体检业务全流程管理

六、目录结构示意

health_parent/
  ├─ health_backend/         # 后端Web接口
  ├─ health_service_provider/# 业务实现
  ├─ health_interface/       # 接口定义
  ├─ health_common/          # 通用模块
  └─ sql/                    # 数据库脚本

七、常用页面入口

  • 体检组管理
  • 会员管理
  • 预约设置
